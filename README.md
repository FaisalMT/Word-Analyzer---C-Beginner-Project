# 🧠 Word Analyzer (C++ Beginner Project)

This is a simple C++ console application built as a beginner programming project.

## 📌 Features
- Reads a word from the user
- Displays the length of the word
- Shows the first and last letter
- Checks if a specific character exists in the word
- Repeats the process if the user wants

## 🛠️ Technologies Used
- C++
- Strings
- Loops
- Conditional statements

## 🎯 Purpose of the Project
This project was created as practice for a Programming 1 course to understand:
- String manipulation
- Loops
- Basic logic building in C++
